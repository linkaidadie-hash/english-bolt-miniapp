> 原项目（**PWA / HTML+JS+SW**）: [github.com/linkaidadie-hash/english-trainer](https://github.com/linkaidadie-hash/english-trainer)
> v2 项目（**微信原生小程序**）: 本仓库

## 一、产品定位

将现有"单词播放器"升级为：

**面向普通用户的免费英语自主学习、自然口语解码与沉浸训练工具。**

核心能力分为五层：

```
大词库
  → 主动记忆
  → 自然口语解码
  → 场景听说训练
  → 全天沉浸复习
```

最终目标不是只做到"认识单词"，而是逐步实现：

```
看得懂
  → 听得懂标准发音
  → 听得懂自然口语
  → 能主动说出来
  → 能在真实场景中使用
```

## 二、首发内容规模

采用"大内容库、小剂量学习"的方式。

| 内容类型 | 数量 |
|---|---|
| 基础单词 | **8,000 ~ 10,000** |
| 高频词组与固定搭配 | 4,000 ~ 6,000 |
| 实用例句 | 15,000 ~ 20,000 |
| 自然口语重点句 | 3,000 ~ 5,000 |
| 场景常用句 | 2,000 ~ 3,000 |
| 场景课程 | 16 个 |
| 场景对话 | 150 ~ 300 组 |
| 分级阅读 | 100 篇 |
| 听力训练组 | 300 ~ 500 组 |
| 口语变化规则 | 100 ~ 200 条 |

### 内容分层

**核心 3,000 词** — 完整配置：
- 英文
- 中文释义
- 词性
- 标准音标
- 标准发音
- 常见搭配
- 2~3 个例句
- 所属场景
- 难度等级
- 词频等级
- 句中常见发音变化
- 听力辨识训练

**扩展 5,000~7,000 词** — 至少包含：
- 英文
- 释义
- 词性
- 音标
- 标准发音
- 一个例句
- 主题和难度标签

> 不是所有一万词都做中文近似音。自然口语重点放在核心词、功能词、高频句和真实场景中。

## 三、自然口语解码模块（核心新增）

> 这是本次大纲新增的核心模块，也是 v2 区别于普通背词小程序的核心特色。

### 3.1 每条重点句需要四层内容

例如：

**书面句子**：`I can help you with that.`

**清晰发音**：`I / can / help / you / with / that.`

**自然美式口语**：`I kən help-yə with that.`

**变化说明**：
- `can` 弱读
- `help you` 连读
- `you` 弱读
- `HELP` 和 `THAT` 是句子重音

> 页面不要只显示音标，而是帮助用户理解：自己期待听到什么，实际听到什么，两者为什么不同。

### 3.2 支持的口语变化类型（首发至少覆盖）

| 类型 | 示例 |
|---|---|
| **弱读** | can → /kən/ ; to → /tə/ ; for → /fɚ/ ; and → /ən/ 或 /n/ ; of → /əv/ 或 /ə/ |
| **缩写** | I will → I'll ; I am → I'm ; I have → I've ; do not → don't ; would have → would've |
| **连读** | turn it on ; pick it up ; help you ; an apple |
| **同化** | did you → 接近 didja ; could you → 接近 couldja ; don't you → 接近 doncha |
| **省音和吞音** | next day ; last night ; friendship ; most people |
| **美式闪音** | water ; better ; city ; pretty ; get it |
| **非正式口语压缩** | want to → wanna ; going to → gonna ; have to → hafta ; let me → lemme ; give me → gimme |

必须标注这是：
- ✅ 自然口语现象
- ✅ 非正式表达
- ❌ 不等于正式书写

### 3.3 三档音频

重点句建议配置：

1. **逐词拆读** — 阶段四 TTS 专项生成
2. **清晰慢速** — `edge-tts --rate=-25%`
3. **自然美式语速** — `edge-tts --rate=+10%` 或神经语音

训练顺序：

```
先听自然语速
  → 猜内容
  → 显示原句
  → 查看口语变化
  → 听慢速拆解
  → 再听自然语速
  → 跟读
  → 隐藏字幕复述
```

> 这比把同一句话机械播放十遍有效得多。

### 3.4 显示开关（设置中可调）

| 设置项 | 默认 |
|---|---|
| 显示标准音标 | ✅ 开 |
| 显示自然口语变化 | ✅ 开 |
| 显示句子重音 | ✅ 开 |
| 显示连读分组 | ❌ 关（初学者可开） |
| 显示中文近似音 | ✅ 开（初学者），熟练后可关 |

**中文近似音只能作为辅助**：

例如：
- `can` 弱读时接近"肯"
- `I'll` 整体接近"爱尔"

> 但不能把整句长期写成中文谐音，否则用户容易依赖中文音系，发音会越学越歪。

## 四、整体开发阶段（8 阶段）

- ✅ 真机预览稳定

详细：[docs/PHASE1.md](PHASE1.md) + [docs/PHASE1.5.md](PHASE1.5.md)

### 阶段二：内容数据层重建
预计 6~10 个有效开发日。

建立统一内容实体：
```
Word / Phrase / Sentence / PronunciationPattern / Scene /
Dialogue / Reading / AudioAsset / Exercise
```

**Word 数据结构**:
```
wordId, word, meaning, partOfSpeech, ipa, frequencyLevel,
difficultyLevel, topics, audioStandard, collocations,
exampleSentenceIds, sceneIds
```

**Sentence 数据结构**:
```
sentenceId, text, translation, difficulty, sceneId, keywords,
standardIpa, naturalIpa, stressWords, speechChunks,
pronunciationNotes, audioSegmented, audioClear, audioNatural,
chineseSoundHint
```

**PronunciationPattern（自然口语规则）**:
```
ruleId, ruleType, sourcePattern, spokenPattern, explanation,
examples, formality, difficulty
```

例：
```
ruleType: weak_form
sourcePattern: can
spokenPattern: /kən/
condition: 非重读、肯定句中
exception: 强调或否定时通常不弱读
```

**验收标准**:
- 8,000 个以上单词可导入
- 3,000 条以上重点口语句可关联
- 所有音频使用真实清单，不猜文件名
- 口语变化可按规则检索
- 后续扩词不需要修改页面代码

### 阶段三：每日主动学习引擎
预计 6~10 个有效开发日。

首页改为"今日训练"。每日训练包含：
- 到期复习
- 新词学习
- 词组学习
- 听音辨词
- 听自然口语辨句
- 中文说英文
- 句子排序
- 跟读模仿
- 错题回炉
- 现实英语任务

**训练不能只考词义**。同一个内容分别记录：
- 是否认识
- 是否听懂标准发音
- 是否听懂自然发音
- 是否会拼写
- 能否主动说出
- 是否能在场景中使用

**间隔复习**：
| 状态 | 复习时机 |
|---|---|
| 完全忘记 | 当天稍后 |
| 模糊 | 1 天后 |
| 基本记住 | 3 天后 |
| 熟练 | 7 天后 |
| 能听会说 | 21 天后 |

> 自然口语听力要单独记录，不能因为用户认识单词，就判定已经掌握听力。

### 阶段四：自然口语解码训练
预计 7~12 个有效开发日。

这是**独立阶段**，不附属于普通单词页。

**核心训练模式**：

1. **听自然语速猜原句** — 只播放自然口语，不显示文字
2. **声音切词** — 播放"Whaddaya wanna do?"，还原"What do you want to do?"
3. **标注变化** — 让用户识别弱读/连读/吞音/闪音/缩写/句子重音
4. **慢速与自然语速对比** — 清晰慢速 ↕ 自然语速
5. **跟读模仿** — 听 → 录 → 回放 → 再录 → 自评（第一版不接付费评分）
6. **隐藏字幕复述** — 听完后隐藏文字，复述

**自然口语专项课程**：
- 弱读入门
- 连读入门
- 美式闪音
- 常见缩写
- 吞音与省音
- 句子重音
- 节奏与意群
- 高频功能词
- 美剧常见压缩表达

**验收标准**:
- 用户能在标准发音与自然发音之间切换
- 每个重点句能显示真实变化原因
- 口语提示不会错误覆盖所有场景
- 中文近似音可以独立关闭
- 至少 3000 条重点句具备自然口语标注
- 高频规则具备例句和训练题

### 阶段五：沉浸与后台播放
预计 5~8 个有效开发日。

后台播放不能只念"apple，苹果"。

**词汇复习频道**：
英文 → 停顿回忆 → 中文释义 → 英文重复 → 常用搭配 → 例句 → 跟读停顿

**自然口语频道**：
自然语速句子 → 停顿猜测 → 清晰慢速 → 中文解释 → 口语变化说明 → 自然语速再播放 → 跟读停顿

**频道建议**:
- 今日已学
- 今日错词
- 自然口语入门
- 弱读专项
- 连读专项
- 美式闪音
- 生活场景
- 旅行场景
- 工作场景
- 外贸场景
- 纯英文沉浸
- 睡前慢速

> 后台播放主要负责：持续接触和复习。
> 主动训练负责：真正理解、回忆和输出。

### 阶段六：场景学习与语言环境
预计 8~12 个有效开发日。

**首发 16 个场景**：

| 生活 | 旅行 | 工作 | 外贸 | 社交 | 紧急 |

| 自我介绍 | 机场 | 工作交流 | 外贸询盘 | 认识新朋友 | 紧急求助 |
| 日常寒暄 | 酒店 | 会议表达 | 客户接待 | | |
| 餐厅点餐 | 问路 | 电话沟通 | | | |
| 商店购物 | 打车 | | | | |
| 看病 | | | | | |

每个场景包含：
- 核心词汇
- 高频词组
- 标准句型
- 自然口语版本
- 短对话
- 听力理解
- 发音解码
- 跟读
- 角色选择
- 隐藏字幕复述
- 通关测试

例：

**书面表达**: `What do you want to order?`
**自然口语**: `Whaddaya wanna order?`
**说明**:
- `what do you` 连读同化
- `want to` 压缩为 `wanna`
- `ORDER` 是主要重音

## 八、技术决策速览

| 决策 | 选型 | 理由 |
|---|---|---|
| 框架 | 微信原生 | 无额外依赖，发布简单，包小 |
| TTS 引擎 | edge-tts (Microsoft Neural) | 免费无限 + 17 个 en-US 神经音 + 三档语速可控 |
| 词库 | ECDICT (CC-BY-SA 4.0) | 商业可用 + 词条全 + 含 IPA |
| 存储 | 微信本地 wx.setStorage | 阶段七不引入数据库 |
| 音频缓存 | 内存 LRU（阶段二） | 避免重复请求 |
| 状态管理 | 简单 Page data + 全局 tts.js | 阶段一足够，不上 Redux/MobX |
| 后台播放 | 微信 wx.getBackgroundAudioManager | 阶段五 |
